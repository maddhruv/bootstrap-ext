Overview

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
This Extension provides another big wing and addition to you websites and applications.

Features

  • Responsive Bootstrap theme
  • Served over fast GitHub CDN
  • Multi-browser and platform support
  • Automatic Cache
  • Easily Customizable
  • Easy to Install

Installation

Download the zip

Download the zipped folder and copy the lib folder contents into your sites' root folder.

Bootstrap Extension CDN

Just use these CDN links

Usage

Colors

All HTML tags and controls elements with .color-text can be styled with the color specified and adding extra .text-lighten-1/5, .text-accent-1/5 or .text-darken-1/5 can lighten or darken the color respectively.

{% highlight html %}

{% endhighlight %}

Background color or Highlight can also be styled by adding .color-bg class to the HTML element. And similarly the color can be lightened or darkened by adding corresponding .lighten-1/5, .accent-1/5 or .darken-1/5 classes to the element.
